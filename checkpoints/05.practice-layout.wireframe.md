# Wireframe: 05.practice-layout + 06.practice-features
# Stories: Practice Page Layout + Interactive Features
# Phase: P1 - UX Design

## COMPONENTS

### 1. Sidebar
- **Description:** Left-side navigation panel with logo, role selector, and nav links
- **States:** default, collapsed (mobile)
- **Responsive:** desktop-visible, mobile-hidden (hamburger trigger)
- **Breakpoint:** 768px

### 2. SidebarLogo
- **Description:** Theory Pass logo at top of sidebar
- **States:** default
- **Responsive:** scales with sidebar width

### 3. RoleSelector
- **Description:** Dropdown/select for role (ADI, Learner, etc.)
- **States:** default, hover, focus, open, disabled
- **Responsive:** full-width within sidebar

### 4. NavLink
- **Description:** Individual navigation link item with icon and label
- **States:** default, hover, active (purple highlight), focus
- **Responsive:** icon-only on collapsed sidebar

### 5. DarkModeToggle
- **Description:** Toggle switch for dark/light mode
- **States:** default (light), toggled (dark), hover
- **Responsive:** maintains position in sidebar

### 6. Breadcrumbs
- **Description:** Navigation trail showing current location
- **States:** default, hover (on links)
- **Responsive:** truncates on mobile with ellipsis

### 7. PageHeader
- **Description:** Main page title "Practice"
- **States:** default
- **Responsive:** font-size reduces on mobile

### 8. SectionHeader
- **Description:** Section title "Study Materials"
- **States:** default
- **Responsive:** font-size reduces on mobile

### 9. MainLayout
- **Description:** Container with sidebar + main content area
- **States:** default, sidebar-collapsed
- **Responsive:** stacked layout on mobile

### 10. Card (Story 06)
- **Description:** Reusable card component for practice/study items with icon, title, description, and action button
- **States:** default, hover (subtle shadow lift)
- **Responsive:** full-width on mobile, adapts to grid on tablet/desktop
- **Breakpoint:** 768px (2 cols), 1024px (3 cols)

### 11. CardButton (Story 06)
- **Description:** Action button within card (e.g., "View Topics", "Start reading", "Start test")
- **States:** default, hover, active, focus, disabled, loading
- **Responsive:** full-width within card

### 12. Modal (Story 06)
- **Description:** Overlay dialog for topic selection with backdrop, title, content area, and close button
- **States:** closed, open, closing (animation)
- **Responsive:** centered, max-width 480px, full-width on mobile with margins

### 13. TopicList (Story 06)
- **Description:** List of selectable topics within the modal
- **States:** default
- **Responsive:** scrollable if content exceeds viewport

### 14. TopicListItem (Story 06)
- **Description:** Individual topic row in the topic list
- **States:** default, hover, focus, selected
- **Responsive:** full-width

### 15. ModalBackdrop (Story 06)
- **Description:** Semi-transparent overlay behind modal
- **States:** default (visible), hidden
- **Responsive:** covers full viewport

}
